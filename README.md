# Diabetes Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a patient has diabetes using machine learning models.

## 📊 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosting
* XGBoost
* Voting Classifier

## ⚙️ Techniques

* Data Preprocessing
* Feature Scaling (StandardScaler)
* Cross Validation
* Hyperparameter Tuning (GridSearchCV)

## 📈 Results

* Best Model: Random Forest (after tuning)
* Accuracy: 77.3%

## 🚀 How to Run

```bash
pip install -r requirements.txt
```

Run the notebook:

```bash
jupyter notebook
```

Pima Indians Diabetes Dataset
